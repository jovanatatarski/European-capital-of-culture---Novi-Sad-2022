# Novi Sad - European Capital of Culture 2022

## Description
This project is a website dedicated to Novi Sad, the European Capital of Culture for 2022. It showcases the city's cultural heritage, events, history, and key landmarks. The website serves as a digital brochure, highlighting the vibrancy and artistic spirit of Novi Sad.

## Features
- **Responsive Design:** Adapted for desktop and mobile viewing with a collapsible navigation menu.
- **Interactive Carousel:** Highlights key cultural institutions and events like the Serbian National Theatre and various festivals.
- **Information Cards:** Detailed cards for major attractions such as EXIT Festival, Petrovaradin Fortress, and Danube Park.
- **Events Page:** Dedicated section for upcoming and past cultural events.
- **History Section:** Overview of Novi Sad's rich history.
- **Contact/Info:** Information page with a contact form.

## Technologies Used
- **HTML5:** Semantic structure of the web pages.
- **CSS3:** Custom styling for layout and visual aesthetics.
- **Bootstrap 4:** Framework for responsive grid layout and components (Navbar, Carousel, Cards).
- **jQuery:** Used for Bootstrap's JavaScript components.

## Setup & Usage
1. Clone or download the repository.
2. Open `index.html` in any modern web browser to view the homepage.
3. Navigate through the links (Events, History, Info) to explore other sections.
